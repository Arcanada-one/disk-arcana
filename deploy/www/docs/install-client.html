Install the client

Linux (x86_64)

curl -fsSL https://disk.arcanada.ai/install.sh | sh

Inspect before running: download install.sh and review it. The installer places disk in /usr/local/bin by default.

Environment: DISK_INSTALL_PREFIX, DISK_VERSION to pin a release tag.

macOS

cargo build --release -p disk-cli
sudo ./scripts/install-macos.sh --binary ./target/release/disk

Or download a release binary from GitHub Releases.

Windows

Download disk-arcana-windows-x86_64.zip from GitHub Releases and follow the Windows E2E runbook in the repository (docs/runbooks/DISK-RB-008-windows-vm-e2e.md).

After install

  • Config: /etc/disk-arcana/disk.toml (Linux) or seeded path on macOS
  • Data: /var/lib/disk-arcana/
  • Local status API: http://127.0.0.1:9444/status (loopback only)
  • Daemon may not start until enrollment completes — expected before first disk enroll
