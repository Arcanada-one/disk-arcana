Getting started

Estimated time: 10–15 minutes for your first synced file.

  1. Create an account

    Open the tenant dashboard, sign up with email and password, or use OAuth.

  2. Verify your email

    Check your inbox for the verification link. The dashboard shows a banner until verification completes.

  3. Install the client

    Follow Install the client. On Linux you can run:

    curl -fsSL https://disk.arcanada.ai/install.sh | sh
  4. Enroll your machine

    An operator issues a one-time enrollment token; you run disk enroll on the device. See Enroll a device.

  5. Initialize a vault share

    On the enrolled device:

    disk share init --preset obsidian-vault

    Pick a preset that matches your folder layout. The vault appears in the dashboard under Vaults.

  6. Confirm sync

    Add a file under the shared path. Within a minute the device Last seen timestamp should update in the dashboard. Check local status:

    curl -sf http://127.0.0.1:9444/status | jq .
