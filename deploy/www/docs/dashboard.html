Use the dashboard

The tenant dashboard at disk.arcanada.ai/dashboard is a single-page app backed by the health HTTP API (:9446 by default). Sign in with email/password or OAuth.

Overview cards

Plan tier, storage quota, enrolled devices, and vault count with usage bars.

Vaults & devices

Lists vault IDs created via disk share init and nodes enrolled via disk enroll. Use Folders on a device row to configure selective sync.

File version history

Browse and restore prior revisions. Retention limits depend on your plan tier.

Vault snapshots

Capture a point-in-time index of all files in a vault and restore later.

Sharing

Create invite links (viewer or editor role). Accept invites via ?sharing_accept=<token> deep link.

Selective sync

Per-device folder prefixes. Empty list = sync entire vault. Enforced on gRPC sync paths.

Recycle bin

Restore, permanently delete, or empty the bin. Expired items are pruned automatically by retention tier.

Conflicts

Unresolved sync conflicts appear with metadata-only resolve actions (no server-side file merge).

Deep links

  • ?sharing=1 — open sharing panel
  • ?selective_sync=1&selective_sync_node=<id> — selective sync
  • ?api=https://host:9446 — override API base (dev)
