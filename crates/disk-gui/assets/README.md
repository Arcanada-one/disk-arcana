# disk-gui/assets

## Icon files

`disk-arcana.icns` and `disk-arcana-placeholder.icns` are **placeholder artwork** —
a simple cat-face silhouette generated by `scripts/gen-placeholder-icon.sh`.
Both files are identical; `disk-arcana.icns` is the one referenced by the
`cargo-bundle` metadata in `Cargo.toml`.

To replace with final artwork:
1. Create or export the finished icon as a 1024×1024 PNG (minimum).
2. Run `scripts/gen-placeholder-icon.sh` with the new artwork, or use
   `iconutil` directly with a full `.iconset` directory.
3. Overwrite `disk-arcana.icns` (and optionally `disk-arcana-placeholder.icns`).
4. Commit.

The placeholder was generated using ImageMagick + `iconutil` on macOS.
It contains 10 resolution entries (16, 32, 128, 256, 512 @1x and @2x).
